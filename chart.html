RESTful Routes

A table of all 7 RESTful routes

Name Path HTTP Verb Purpose Mongoose Method
Index /campgrounds GET Display a list of all the Campgrounds Campground.find()
New /campgrounds/new GET Display a form to make new campground N/A
Create /campgrounds POST Add new campground to DB, then redirect somewhere Campground.create()
Show /campgrounds/:id GET Show info about one specific campground Campground.findById()
Edit /campgrounds/:id/edit GET Show edit form for one dog Campground.findById()
Update /campgrounds/:id PUT Update particular campground, then redirect somewhere Campground.findByIdAndUpdate()
Destroy /campgrounds/:id DELETE Delete a particular camground, then redirect somewhere Campground.findByIdAndRemove()
